GitHub token

Required to save your clearings back to GitHub. Stored only in this browser.

How to get a token

1. Go to github.com → your avatar → Settings

2. Scroll to Developer settings → Personal access tokens → Tokens (classic)

3. Click Generate new token (classic)

4. Name it Earthkeeper, tick the repo scope

5. Click Generate, copy the token, paste it above

How data is stored

Location data: data/locations-[region].json

Your progress: data/progress.json

Both are plain files in your GitHub repo — they never expire, never pause, and you can read them any time.
